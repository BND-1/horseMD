# 公式与 Mermaid

<span class="version-badge">适用于 HorseMD v0.6.5</span>

## 数学公式

行内公式使用单个美元符号：

```md
质能方程是 $E = mc^2$。
```

独立公式使用成对的 `$$`：

```md
$$
f(x) = \int_{-\infty}^{\infty} \hat f(\xi)e^{2\pi i\xi x}\,d\xi
$$
```

HorseMD 使用 KaTeX 渲染。过宽公式会在自身区域横向滚动，不会撑破正文宽度。单行 `$$x^2$$` 也会被安全规范化为显示公式，但分行写法更容易在其他 Markdown 工具中兼容。输入 `$$` 后回车，或使用 `/math`，光标会留在公式源码中并允许连续输入；离开编辑后恢复紧凑预览，点击工具栏“编辑代码”可再次修改。

输入行内公式时会显示实时预览。也可以先输入两个 `$`，再把光标移到中间补写内容；输入期间保持源码可编辑，光标离开后自动渲染。点击已有行内公式会打开编辑框，修改时预览同步更新，确认后写回正文；编辑框里的“清空”可以快速删掉当前公式内容。

默认情况下，光标紧挨行内公式时按 Backspace 或 Delete 会先选中整段公式，再按一次才真正删除，避免误删完整公式。想恢复一键删除，可以在“设置 → 编辑器 → 行内公式删除”里切换为“快速”。

## Mermaid 图表

创建语言为 `mermaid` 的围栏代码块：

````md
```mermaid
flowchart LR
  A[打开文件] --> B[开始写作]
  B --> C[保存]
```
````

富文本模式默认显示渲染后的图表，并保留源代码编辑入口。点击图表会进入全屏灯箱，并严格按照图表原始宽高比显示，横向或纵向长图不会被塞进固定方形画布。

灯箱顶部提供缩小、当前倍率、放大、适应窗口和 `1:1` 原始尺寸按钮。也可以按住拖动进行平移，使用 `Ctrl+滚轮` 在 0.2～10 倍之间缩放，按 Escape 或点击背景关闭。普通图片灯箱使用同一套比例和缩放控制。

Mermaid 语法错误时保留源码供修改，不会执行图表中的脚本。
