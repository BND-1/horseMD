# 文字与段落格式

<span class="version-badge">适用于 HorseMD v0.6.0</span>

HorseMD 支持直接输入 Markdown 语法，也支持选区工具栏、快捷键和斜杠菜单。

## 段落类型

在空段落开头输入以下内容并按空格：

| 输入 | 结果 |
| --- | --- |
| `# `～`###### ` | 一到六级标题 |
| `- ` 或 `* ` | 无序列表 |
| `1. ` | 有序列表 |
| `- [ ] ` | 任务列表 |
| `> ` | 引用 |
| `` ``` `` | 代码块 |

也可以输入 `/` 后选择对应块。`Ctrl+1`～`Ctrl+6` 将当前段落转换为标题，`Ctrl+0` 转回正文。

## 文字格式

选中文字后会出现浮动工具栏，可设置粗体、斜体、删除线、高亮、行内代码和链接。常用快捷键包括 `Ctrl/Cmd+B`、`Ctrl/Cmd+I`，黄色高亮使用 `Ctrl/Cmd+Alt+H`。

Markdown 源码中的 `**粗体**`、`*斜体*`、`` `代码` ``、`~~删除线~~` 和链接语法会在富文本中即时显示。切换到源码模式可以检查实际标记。

行内代码既可以按顺序输入，也可以连续输入两个反引号后直接填写中间内容。编辑已有行内代码时，点击代码文字的右边缘即可继续追加；在代码外点击后输入的仍是普通正文。

正文结束后仍会保留一段舒适的滚动留白。点击正文下方的这段空白，会把光标直接放到文档末尾，可以继续输入，不必精确点击最后一行。

::: tip 粘贴 Markdown
在普通正文中粘贴包含标题、列表或表格的 Markdown，会按结构解析；在代码块中粘贴则保持原始文本，不会被重排。
:::

从微信公众号等网页编辑器复制富文本时，HorseMD 会把用于排版的 `section`、`div` 视觉段落转换为正常段落，同时保留加粗、斜体、列表等可识别格式，避免多段内容挤成一段。
