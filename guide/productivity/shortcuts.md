# 快捷键

<span class="version-badge">适用于 HorseMD v0.6.5</span>

表格中的 `Ctrl/Cmd` 表示 Windows 使用 Ctrl、macOS 使用 Command。

可以在 **设置 → 键盘快捷键** 中修改大部分应用快捷键。点击某个快捷键按钮后直接按新的组合键即可录制；录制时按 `Esc` 取消，按 `Delete` 清空。若新快捷键已经被其他命令占用，HorseMD 会提示冲突，不会静默覆盖。

![键盘快捷键设置](/images/v0.6.5/keyboard-shortcuts.png)

加粗、斜体、高亮等编辑器原生命令当前保持固定，以避免影响输入法、表格、代码块和编辑器插件行为。标题层级与正文切换已经支持自定义。

## 自定义快捷键

1. 点击左侧齿轮打开 **设置**。
2. 进入 **键盘快捷键**。
3. 用搜索框按命令名称、分类、命令 ID 或快捷键查找目标命令。
4. 点击当前快捷键按钮，然后直接按新的组合键。
5. 需要取消录制时按 `Esc`；需要让某个命令不再响应快捷键时按 `Delete` 或点 **清空**。
6. 点单行的 **恢复默认** 只恢复这一条；点顶部 **全部恢复默认** 会清空所有自定义。

HorseMD 会阻止普通单字母、`Enter`、`Tab`、`Esc`、复制、粘贴、撤销、退出等保留键。设置页打开时，保存、查找、侧边栏等文档快捷键不会在后台误触发。

## 文件与标签

| 操作 | 快捷键 |
| --- | --- |
| 新建文档 | `Ctrl/Cmd+N` |
| 打开文件 | `Ctrl/Cmd+O` |
| 打开文件夹 | `Ctrl/Cmd+Shift+O` |
| 保存 | `Ctrl/Cmd+S` |
| 另存为 | `Ctrl/Cmd+Shift+S` |
| 关闭标签 | `Ctrl/Cmd+W` |
| 下一个标签 | `Ctrl+Tab` |
| 上一个标签 | `Ctrl+Shift+Tab` |

## 编辑与导航

| 操作 | 快捷键 |
| --- | --- |
| 命令面板 | `Ctrl/Cmd+P` |
| 查找 | `Ctrl/Cmd+F` |
| 加粗 | `Ctrl/Cmd+B` |
| 切换侧边栏 | `Ctrl/Cmd+Shift+B` |
| 切换大纲 | `Ctrl/Cmd+Shift+L` |
| 切换源码模式 | `Ctrl/Cmd+/` |
| 切换主题 | `Ctrl/Cmd+Shift+T` |
| 标题 1～6 | `Ctrl/Cmd+1`～`Ctrl/Cmd+6` |
| 转为正文 | `Ctrl/Cmd+0` |
| 黄色高亮 | `Ctrl/Cmd+Alt+H` |

## 输出

| 操作 | 快捷键 |
| --- | --- |
| 导出 PDF | `Ctrl/Cmd+Shift+E` |
| 复制 | `Ctrl/Cmd+C` |

`Ctrl/Cmd+B` 用于加粗；`Ctrl/Cmd+Shift+B` 用于切换侧边栏。
