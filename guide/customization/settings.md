# 设置说明

<span class="version-badge">适用于 HorseMD v0.6.5</span>

点击活动栏左下角齿轮打开设置。设置以临时标签呈现，离开后不会像普通文档一样在下次启动时恢复。

![HorseMD 设置页](/images/v0.6.5/settings.png)
<span class="image-caption">设置项按编辑器、外观、文件、键盘快捷键和关于分组。</span>

## 编辑器样式

调整文档字体、代码字体、字号、行距、段距和页宽。右侧小型文档预览会实时显示效果；页宽也可以从状态栏快速切换窄、中、宽和全宽，并通过滑块微调。

源码模式默认跟随文档正文的字号，也可以在“源码模式”区域单独调大或调小。需要更细的视觉定制时，可以在同一页的“自定义 CSS”中写 CSS；它会叠加在当前主题之上，并作用于真实编辑器和设置页预览。预览包含标题、强调文字、链接、行内代码、快捷键、引用、列表、任务清单、表格和代码块，适合先检查常见选择器。

自定义 CSS 可分为多个命名片段，每个片段能独立开关、重命名、排序或删除；后面的已启用片段会覆盖前面同名的规则。暂时切换到其他文档时，回到设置会继续打开刚才编辑的片段，不必重新找到它。

## 外观

选择内置主题或导入兼容的 Typora CSS 主题。主题主要改变应用外壳和编辑区的整体颜色；如果只想微调标题、引用、行内代码等文档细节，优先使用“编辑器 → 自定义 CSS”。

## 文件与键盘

文件设置中可以开启“显示隐藏文件”，文件树会显示 `.claude`、`.cursor` 等隐藏目录，但 `.git` 和 `node_modules` 始终隐藏，避免噪声和性能问题。键盘快捷键设置支持录制大部分应用命令的快捷键，并会提示冲突和保留键。

## 校对与语言

英文拼写检查默认关闭，开启后只作用于富文本正文，查找框、文件名和源码输入不会出现拼写红线。语言可在中文和 English 之间切换。

## 图床与关于

图床区域配置自定义上传命令；关于区域显示实际应用版本，并提供官网、GitHub、Gitee 和检查更新入口。

大部分偏好保存在本机 `horsemd.settings.v1` 中。重装应用前如需完全保留个性化设置，应同时备份系统用户数据目录。
