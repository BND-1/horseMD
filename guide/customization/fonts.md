# 文档与代码字体

<span class="version-badge">适用于 HorseMD v0.6.5</span>

打开“设置 → 编辑器”，顶部有两个独立选择器：

- 文档字体：影响正文和标题。
- 代码字体：影响围栏代码块、CodeMirror 和行内代码的等宽显示。

![HorseMD 设置页中的字体和排版选项](/images/v0.6.5/settings.png)
<span class="image-caption">选择器、字号、行距、段距、页宽和自定义 CSS 会在右侧小型文档中实时预览。</span>

## 选择字体

点击字段后可以直接输入搜索本机字体；字体列表异步加载完成后，正在输入的关键词和焦点都会保留。字体列表优先显示完整字体名，过长时可悬停查看系统提示；鼠标悬停时，设置预览和当前编辑器会临时应用该字体。点击后正式保存，选择“默认”恢复系统字体栈。

浏览器会请求本地字体访问权限，该权限只用于枚举字体名称和预览，不会读取文档或把字体上传到网络。

## 字号与间距

同一页还可以调整正文的字号、行间距、段落间距和页面宽度。更改会立即应用到当前文档和右侧预览。

- 行间距同时影响普通段落、无序列表和有序列表的每一项。
- 段落间距同时影响列表前后留白、列表项之间的间距和嵌套列表的留白；默认比例保持不变。
- 富文本围栏代码块的前后留白也跟随段落间距；代码块内部的语言选择和复制操作不受影响。
- 这些排版选项只改变显示，不会改写 Markdown 源码。

## 字体没有出现

确认字体已经安装到系统并允许当前用户使用，然后重新打开设置。某些字体只有单一字重，标题粗体可能由系统合成。代码字体建议选择明确支持 ASCII 标点的等宽字体。

设置页底部提供方正字库和 Nerd Fonts 外部链接；字体许可由对应提供者决定。

## 源码字号和自定义 CSS

源码模式默认使用和正文一致的字号。如果觉得源码太小，可以在“设置 → 编辑器 → 源码模式”里把源码字号调大。

“自定义 CSS”位于同一页，适合微调标题颜色、引用样式、行内代码字号等文档细节。它会叠加在当前主题之上；如果想整体替换主题风格，仍建议使用“设置 → 外观”导入主题。
