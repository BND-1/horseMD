字体资源

为 HorseMD 寻找并安装更多字体

推荐字体站

Google Fonts
海量免费字体（含中文：Noto Sans SC、ZCOOL 等），可预览、筛选、下载
Nerd Fonts
代码字体 + 图标，适合「代码字体」设置
Fontshare
高质量免费字体（手写、衬线、显示体）
猫啃网
中文免费可商用字体合集

推荐字体

  • 中文：思源黑体 / 思源宋体、霞鹜文楷、未来荧黑
  • 英文：Inter、Lora、Crimson Pro
  • 代码：JetBrains Mono、Fira Code、Cascadia Code

安装方法

macOS：下载字体文件（.ttf / .otf），双击 → 安装；或拖到 ~/Library/Fonts。

Windows：下载 .ttf，右键 →「为所有用户安装」。

安装字体后重启 HorseMD，新字体才会出现在字体选择器中。
