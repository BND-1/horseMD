获取更多字体

装好系统字体后重启 HorseMD，就能在「设置 → 排版」的字体选择器里选到。

推荐字体站

Google Fonts 海量免费英文字体，可按风格筛选，下载 .ttf Nerd Fonts 代码字体 + 图标（适合「代码字体」，终端风，含大量连字与符号） Fontshare 高质量免费商用字体（含手写、衬线、显示体） 猫啃网 中文免费可商用字体合集（思源、霞鹜文楷、未来荧黑 等）

几款推荐

  • 中文文档：思源黑体 / 思源宋体、霞鹜文楷、未来荧黑
  • 英文文档：Inter、Lora、Crimson Pro
  • 代码：JetBrains Mono、Fira Code、Cascadia Code，或任意一个 Nerd Font

怎么装

macOS：下载字体文件（.ttf / .otf），双击 → 弹出「字体册」点「安装字体」；或把文件拖到 ~/Library/Fonts（仅当前用户）或 /Library/Fonts（全体用户）。

Windows：下载 .ttf，右键 →「为所有用户安装」（推荐，避免只在当前会话生效）。

⚠️ 装完字体后重启 HorseMD（退出再打开），新字体才会出现在字体选择器里（选择器在打开时枚举一次系统字体）。
