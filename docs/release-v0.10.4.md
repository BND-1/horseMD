# HorseMD v0.10.4

本版本继续打磨日常写作、排版和桌面端同步体验。重点包括更完整的自定义 CSS 工作流、移动端只读保护、右侧悬浮章节导航，以及表格、公式和 PDF 导出的稳定性修复。

## 新功能

### 可组合的自定义 CSS

“设置 > 编辑器 > 自定义 CSS”现在支持多个具名样式片段。

- 每个片段可单独启用、排序、重命名或删除；启用项按列表顺序叠加。
- 旧版单段 CSS 设置会自动迁移，不丢失既有内容。
- 桌面端可通过受限的“检查元素”操作查看真实文档选择器，便于定位样式目标，不会向渲染层开放 Node 权限。
- 设置预览补全了常见 Markdown 元素：标题、粗体、斜体、删除线、链接、行内代码、键盘按键、引用、列表、任务列表、表格和代码块。
- 从其他文档返回设置后，会继续停留在刚才正在编辑的 CSS 片段。

### 悬浮章节导航

桌面端含标题的文档会在右侧显示低干扰的章节指示器。

- 悬停或使用键盘聚焦时展开可滚动的标题列表。
- 当前章节会随阅读位置更新；点击标题可平滑跳转，富文本和源码模式均可使用。
- 长标题会省略显示，不会挤压正文宽度。
- 点击跳转后、或鼠标移出导航区域后，面板会自动收起。

### 移动端只读模式

iOS 和 Android 顶部新增只读锁定开关。

- 开启后不能修改富文本、源码、代码块或通过粘贴、拖放、撤销/重做改动文档。
- 仍可滚动、选择、复制和打开链接，方便移动阅读时避免误触。
- 桌面端行为不变。

### 同步连接 User-Agent

WebDAV 和 S3 连接现在可选填 User-Agent，满足部分服务商的客户端标识要求。

- 此项与连接的公开配置一起保存，不会写入加密的密码或 S3 Secret。
- 留空时维持原有连接行为。

## 改进

### 表格编辑与阅读

- 内容较少的表格保持自然宽度，不再默认铺满正文。
- 超宽表格仅在确有需要时横向滚动，并使用更清晰的主题化表面区分正文。
- 列边缘悬停可显示添加行/列入口；按住列边界后进入细线、实时的列宽预览，松开后保存最终宽度。
- 在横向滚动到最右侧时操作列边缘，不会再把表格跳回最左边。

### 公式与 PDF

- 显示公式仅在实际超出可用宽度时出现水平滚动区域；正常宽度的公式不会显示多余的滚动控件。
- 带 `\\tag{...}` 的显示公式为编号预留独立空间，避免富文本预览中公式与编号重叠。
- 极长行外公式导出 PDF 时会在可断点处分行，避免被裁切或整体缩小到难以阅读。
- 无图片文档导出 PDF 时，不再因字体就绪时间而错误提示“部分资源可能未完整加载”。

### 大文档与编辑器交互

- 代码块不再受不稳定的高度估算影响，滚动到代码块附近或选中代码块时更稳定。
- 已补充源码/富文本连续切换中的精确光标映射回归覆盖，包括表格单元格、段落、列表和代码块。
- 作者在段落中使用的内联 `<font>`、`<span>` 等 HTML 样式，不会再在正文位置误触发块操作拖拽柄；块操作仍保留在正文左侧操作区。

## 修复

- 修复 Windows 命令面板在悬停或滚动结果时可能发生的合成闪烁（[#62](https://github.com/BND-1/horseMD/issues/62)）。
- 修复 H5/H6 英文标题被强制转为大写，保留作者原有大小写（[#63](https://github.com/BND-1/horseMD/issues/63)）。
- 修复反复插入表格行/列后内容写入相邻单元格、生成空行/列，或保存、重开后未编辑单元格出现多余 `<br>` 的问题；有意添加的单元格内换行仍会保持（[#86](https://github.com/BND-1/horseMD/issues/86)）。
- 修复宽表格右侧列控件和列宽调整在横向滚动后的定位与状态问题（[#86](https://github.com/BND-1/horseMD/issues/86)）。

## 安装

发布时将提供以下桌面安装包：

| 平台 | 安装文件 | 架构 |
| --- | --- | --- |
| macOS | `HorseMD-0.10.4-arm64.dmg` | Apple Silicon |
| macOS | `HorseMD-0.10.4.dmg` | Intel x64 |
| Windows | `HorseMD-Setup-0.10.4.exe` | x64 |
| Linux | `horse_0.10.4_amd64.deb` | x64 |

> 桌面安装包暂未签名。Windows 出现 SmartScreen 提示时，选择“更多信息 -> 仍要运行”；macOS 可以右键应用选择“打开”，必要时执行：
>
> ```bash
> xattr -dr com.apple.quarantine /Applications/HorseMD.app
> ```
